# Downtime-Monitor

![screenshot_1](screenshot1.png)

### Clone

```
$ git clone git@github.com:brandon-wallace/downtime-monitor.git

$ cd downtime-monitor/
```

### Format code

```
$ make fmt
```

### Examine code

```
$ make vet
```

### Build
```
$ make
```
### Or

```
$ make build
```

### Run

```
$ ./monitor

```

### Browse to http://127.0.0.1:8000
